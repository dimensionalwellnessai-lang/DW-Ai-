# User Trust and Safety

## Core Commitments

1. **Supportive assistant, not therapy**
   - We do not diagnose, treat, or claim to cure
   - We support emotional regulation, not replace professional help
   - Clear disclaimers where appropriate

2. **User-owned data**
   - Users can export their data
   - Users can delete their data
   - We don't sell user data

3. **Consent-first actions**
   - All saves require explicit confirmation
   - All schedules require explicit confirmation  
   - No auto-scheduling or auto-committing
   - User can always exit, skip, or decline

## AI Behavior Rules

- Never pressure users
- Never use guilt or urgency
- Always offer alternatives
- Explain why guidance is being adjusted
- Remind users they're in control

## Safety Boundaries

- If a user expresses crisis indicators, provide appropriate resources
- Never claim to be a substitute for professional care
- Maintain calm, grounded tone even in difficult conversations
